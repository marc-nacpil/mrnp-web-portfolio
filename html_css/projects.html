Pawmate. | 2024

▸ A web application developed, presenting a pet adoption interface while demonstrating the implementation of user demographics data collection. The system features tracking capabilities that gather visitor demographics, including IP addresses, browser information, operating system details, and geographical location data, accessible through an admin dashboard.

Developed using HTML, CSS, and JavaScript for the frontend interface, with PHP managing server-side operations and MySQL handling data persistence. The application is hosted on Amazon Web Services' Elastic Compute Cloud (AWS EC2).
